🛡️ CyberShield-AI
AI-Powered Cyber Threat Detection & Log Analysis Tool

CyberShield-AI is a desktop-based cybersecurity tool designed to detect and analyze cyber threats in real time. It combines machine learning, threat intelligence, and log analysis to help identify malicious URLs, suspicious IPs, and abnormal system activities.

🚀 Features
  🔗 URL Threat Scanner
  - Detects phishing & malicious websites
  - Hybrid ML + rule-based analysis

  📊 CSV Log Analyzer
  - Identifies brute-force attacks
  - Detects suspicious IP activity
  - Works with flexible log formats

  🧠 Threat Explanation Engine
  - Explains WHY something is dangerous
  - Provides real-world impact + recommendations

  📄 Report Generator
  - Export security reports (PDF)

  💻 Desktop Application
  - Built using CustomTkinter
  - Packaged as executable (.exe)

🧰 Tech Stack
- Python
- Scikit-learn
- CustomTkinter
- SQLite
- ReportLab
- PyInstaller

**▶️ Run Locally**

# Clone repository
git clone https://github.com/YOUR_USERNAME/CyberShield-Desktop.git

# Navigate to folder
cd CyberShield-Desktop

# Install dependencies
pip install -r requirements.txt

# Run application
python main.py


📦 Build Executable
pyinstaller --onefile --windowed --icon=assets/icon.ico main.py
